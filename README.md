# WikipediaGame
Ce projet s'inspire du jeu de la course a la page wikipedia

## Le but du jeu

Ce jeu consiste en partir d'une page wikipedia au hasard et aller à une autre page au hasard en passant seulement par les href dans les pages

# Installation
Pour jouer a ce jeu, vous devez aller dans le dossier `server`. Installer les dépendances puis lancer jeu avec la commande `node index.js`
